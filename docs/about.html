About This Template

A starter template for web development mini-projects.

Stack

  • Python Flask — lightweight WSGI web framework
  • SQL SQL — ORM with SQLite (swappable to PostgreSQL/MySQL)
  • JS Vanilla JavaScript — async fetch API, DOM manipulation
  • HTML Jinja2 templates — server-side rendering with template inheritance
  • CSS Custom CSS — responsive layout with CSS variables

Quick Start

# 1. Clone the repository
git clone https://github.com/UCA-ESTK-Pr-BERKA/Gxx-202526.git
cd Gxx-202526

# 2. Create and activate a virtual environment
python -m venv MPWeb
source MPWeb/bin/activate      # Windows: MPWeb\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run the application
python run.py

# 5. Open http://127.0.0.1:5000 in your browser
